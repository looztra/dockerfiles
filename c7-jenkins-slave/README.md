What?
=====

* A docker container based on Centos (7), ready to run as a jenkins slave with java/maven/git/mercurial/docker.
* Built with Ansible.

Tools
=====

* openjdk6
* openjdk7
* openjdk8
* maven 3.0.5
* maven 3.1.1
* maven 3.2.5
* maven 3.3.3
* git
* mercurial
* nodejs + npm
* docker client
